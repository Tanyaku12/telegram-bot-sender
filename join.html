Buka Grup via ID

ubah chat id jadi link t.me
Masukkan id grup/channel (mis. -1003976179139) atau @username → dapat link t.me yang langsung membuka grupnya di Telegram.
Diisi otomatis dari halaman kirim pesan. Kalau diisi & bot ada di grup itu, halaman ambil nama grup + link undangan resmi via getChat.
Grup/channel privat berformat -100…. Link privat (t.me/c/…) hanya membuka grup bagi yang sudah jadi anggota.
